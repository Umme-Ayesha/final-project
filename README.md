# Personal To-Do List Application

A simple command-line application to manage your tasks.

## Features

- Add tasks with title, description, and category
- View all tasks
- Mark tasks as completed
- Edit existing tasks
- Delete tasks
- Automatic saving of tasks to a JSON file
